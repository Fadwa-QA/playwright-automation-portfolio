# Playwright Automation Portfolio (Python + Pytest)

## Overview
UI automation project using:
- Python
- Playwright
- Pytest
- Page Object Model (POM)

## Site Under Test
https://practicetestautomation.com/practice-test-login/

## Scenarios Covered
- Successful login
- Invalid login
- Empty username
- Empty password
- Logout

## Setup
1) Install dependencies:
pip install -r requirements.txt

2) Install Playwright browsers:
playwright install

## Run Tests
pytest -v -s
